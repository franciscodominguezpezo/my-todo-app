# 📅 Agenda de Eventos - React + Supabase

Una aplicación de gestión de eventos moderna y funcional construida con **React 18**, **TypeScript** y **Vite**, utilizando **Supabase** como Backend-as-a-Service (BaaS).

## 🚀 Características

- **CRUD Completo:** Crear, Leer, Actualizar y Borrar eventos en tiempo real.
- **Categorización:** Clasificación de eventos por categorías (Deportivos, Formación, Trabajo, Médicos).
- **Backend Seguro:** Implementación de políticas de seguridad **RLS (Row Level Security)** en Supabase.
- **Interfaz Responsiva:** Diseño limpio utilizando **Bootstrap 5**.
- **Experiencia de Usuario:** Notificaciones interactivas y diálogos de confirmación con **SweetAlert2**.
- **Tipado Robusto:** Uso estricto de interfaces de TypeScript para mayor seguridad en el desarrollo.

## 🛠️ Stack Tecnológico

- **Frontend:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Base de Datos:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Estilos:** [Bootstrap 5](https://getbootstrap.com/)
- **Alertas:** [SweetAlert2](https://sweetalert2.github.io/)
